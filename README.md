# 穿搭分身 / StyleTwin（暂定名）

一款 iOS 穿搭 App：上传穿搭照片获取 **AI 图文点评**，按需生成 **风格化"角色卡"**，把卡中服装 **拆解成单品** 存入 **数字衣柜**，再用 **2D 换装** 给虚拟形象搭配。

## 产品定位（MVP 决策）
- **A 方向：风格化虚拟形象**——形象为美化的数字分身，不追求与真实体型一致。
- 拆解走 **「生成角色卡 → 分割切件」** 两段式（把难分割变易分割）。
- 换装走 **2D 纸娃娃 + 可选"上身重生成"**，**不做 3D、不做 AR 全身试穿**（本期）。
- 生成依赖 **云端算力**，人像数据 **最小留存 + 明示同意 + 合规优先**。
